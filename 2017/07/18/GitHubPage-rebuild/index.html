迁移与改版

之前用Jekyll搭建博客（github官方推荐），当时参考中文文档 Jekyll • 简单静态博客网站生成器 - 将纯文本转换为静态博客网站 以及英文文档 Home · Run Jekyll on Windows ，费了很长的时间在本地配置配置好了环境（jekyll不推荐在windows下配置环境），相比之下，Hexo要容易得多。而具体差别，又或是其他各种博客程序的比较，大可以去知乎上作进一步了解：
FarBox、Jekyll、Octopress、ghost、marboo、Hexo、Medium、Logdown、prose.io，这些博客程序有什么特点？
个人博客已经几个月没有维护，之前的样式设计（真算不上设计）算不上丑，但自己不太喜欢，就想从头再来，重做一个博客，并且选用Jekyll以外的博客程序来做，而最终选择了Hexo。
